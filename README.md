# 云谷高中·凌云大富翁地图编辑器

一个基于 React 18 + TypeScript + Vite + Tailwind CSS 构建的教育游戏化地图编辑器。产品围绕云谷高中真实《凌云积分制度》设计，让学生可以通过拖拽方式自由配置大富翁棋盘，并导出为 PNG、PDF、拼图版 PDF 或 JSON。

## 项目简介

该项目将学校制度中的正向积分激励、PA 任务、T 记录惩罚与消 T 修复机制进行桌游化表达。编辑器支持：

- 以经典方形大富翁方式编辑棋盘外圈格子
- 使用七维素养颜色体系表达 FC / AC 地产格
- 通过属性面板快速修改格子名称、描述、分值与等级
- 导出为适合展示、存档、打印的多种格式

## 如何安装运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## 功能清单

### 已完成

- 顶部导航栏：新建地图、导入 JSON、规则说明、保存 JSON
- 左侧组件库：特殊格 + 七维素养 FC / AC 拖拽卡片
- 中央棋盘：经典方形布局、空格显示、选中高亮、中央雷达图
- 右侧属性面板：实时编辑名称、描述、积分、素养颜色和 FC / AC
- Step 6 拖拽功能：使用原生 HTML5 Drag and Drop，支持覆盖放置与 drop target 高亮
- Step 7 雷达图：SVG 绘制七维雷达图，配合静态假数据
- Step 8 JSON 导入导出：完整保存与恢复棋盘配置
- Step 9 PNG / PDF 导出：仅截取棋盘主体导出
- Step 10 拼图版 PDF：多页 A4 分割、页码、虚线切割框、四角对齐标记
- Step 11 规则说明弹窗：内置完整规则说明与关键表格
- Step 12 最终优化：空状态、基础错误处理、loading 状态、响应式布局与统一视觉规范

### 交互说明

- 从左侧组件库把卡片拖到棋盘格子即可完成放置
- 拖拽到已有内容的格子时会直接覆盖
- 放置完成后，右侧面板会自动切换到对应格子
- 点击底部导出栏可导出当前棋盘
- 点击顶部或底部“导入 JSON”均可恢复已有配置

## 技术栈说明

- **React 18**：组件化 UI 构建
- **TypeScript**：严格模式下保证类型安全
- **Vite**：开发与构建工具
- **Tailwind CSS**：设计 Token 与统一样式系统
- **SVG**：雷达图与棋盘矢量友好显示
- **原生 HTML5 Drag and Drop**：实现拖拽放置
- **html2canvas**：棋盘截图导出 PNG / PDF
- **jsPDF**：导出打印版 PDF 与拼图版 PDF
